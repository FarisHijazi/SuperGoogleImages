### The Image Panel

There is one big panel containing 3 sub-panels, you only get to see one sub-panel at a time.
Moving to the next mainImage will rotate the 3 sub-panels, and you will visit each one as they rotate.
Main panel selector: `#irc_cc`
Sub-panels contain a `data-ved` attribute, only the active one will have data, the other 2 will be `null`.
Active sub-panel selector:    `#irc_cc div.irc_t:not([data-ved="null"])`
Each panel has a mainImage `a.irc_mil img.irc_mi` (the one that we see), and the mainThumbnail `a.irc_mutl img.irc_mut` (which is used when the mainImage didn't load yet)

You can access the panel's current itemId value from the dataset field `panel.dataset` which is an object of the format: `{"itemId": string, "ved": string}`

There is a hidden div that has a table to convert `data-ved` to `index-id`/`iid` (which looks like `i3849`).  
Selector: panel.`.irc_rcc div[data-ved][id]`
